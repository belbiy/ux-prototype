Panels - Accordion

Panels are Bootstrap content containers that you can expand and collapse. There are "accordion" and "standard" versions of these panels.

Notes:
The panel heading div has been coded to operate the expand/collpase action. Use the div structure in this example for your panels. Be sure to set a class of "panel-heading" on controlling div. Also, the controlling div will have an href. Use this href to target the panel you wish to connect, match the href to the target div id. Be sure to name a unique ID for each new panel you create on your page.

How to set a panel to open by default on page load:
If you want to open a panel by default add the class of "in" to the targetted panel-body container <div id="collapseOne" class="panel-collapse collapse in">. Also change expander-collapsed to expander-expanded to point the arrow properly for page load.

Accordion versus Standard Panels:
To make the panels behave as an accordion a "data-parent" must be added to each controlling div. Set the "data-parent" equal to the DIV that contains all the panels. Please see Bootstrap JavaScript Reference material for COLLAPSE for more information.

Author: Keith Howard
